## A collection of small scripts for managing waypoints in OnX Hunt map.

- Convert-PotaToOnxKml.ps1:

Exports a onX-compatible GPX file with a list of POTA parks, including custom icon and coloring based on activation status.  Requires both the .csv and the .gpx export from pota.app.

- Add POTA to digial log.ps1:

Not actually directly related to onX, but a script which takes a digital log in .adi format and adds the POTA park information, such that it can be directly submitted.
